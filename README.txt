# Tenacity-Tutors-Admin
Is a mobile app platform that allow students to find a tutor of their desired subject.
The application consist of video calling feature which is integrated with Jitsi SDK.

*Student signs up and login to the app
*Updates profile
*View communications sent by the admin
*Sends and recieve direct message from the admin (Chat system)
*Request a tutor for a desired subject, dates and time.
*Joins a session with the tutor that request was assigned to.
